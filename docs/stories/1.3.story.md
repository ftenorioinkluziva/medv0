## Story

**Como** paciente,
**Quero** preencher meus dados básicos de saúde,
**Para** que os agentes tenham contexto mínimo para personalizar análises.

---

## Contexto

O Medical Profile é o coração do contexto pessoal do paciente. Dados básicos incluem antropometria, pressão arterial, FC em repouso e objetivos de saúde. Baseado no schema real do sistema anterior (campo-a-campo validado).

---

## Acceptance Criteria

### AC1 — Schema `medical_profiles` (dados básicos)

Colunas obrigatórias:
- `age` (integer)
- `gender` (text: masculino/feminino/outro)
- `height` (integer, cm)
- `weight` (numeric, kg)
- `systolic_pressure` (integer, mmHg)
- `diastolic_pressure` (integer, mmHg)
- `resting_heart_rate` (integer, bpm)
- `health_objectives` (text)

Colunas opcionais (nullable):
- `medical_conditions` (text[] — array de condições)
- `medications` (text[] — array de medicamentos)
- `allergies` (text[] — array de alergias)
- `surgeries` (text[] — array de cirurgias)
- `family_history` (text)
- `notes` (text)

### AC2 — Biomarkers opcionais (laboratoriais)

Colunas nullable:
- `handgrip_strength` (numeric, kgf)
- `sit_to_stand_time` (numeric, segundos)
- `vo2_max` (numeric)
- `body_fat_percentage` (numeric)
- `co2_tolerance_test` (numeric)
- `latest_biomarkers` (JSONB — snapshot do último exame processado)
- `biomarkers_updated_at` (timestamp)

### AC3 — Server Action de upsert

- `upsertMedicalProfile(data)` em `src/lib/actions/medical-profile.ts`
- Validação Zod com campos obrigatórios e opcionais bem tipados
- `INSERT ... ON CONFLICT (user_id) DO UPDATE`
- Autorização: apenas o próprio usuário pode salvar seu perfil

### AC4 — Formulário básico

- Tela `/app/profile` com seção "Dados Básicos"
- Shadcn UI: Form, Input, Select, Textarea
- Feedback visual de salvamento (toast de sucesso/erro)
- Pre-popula campos com dados existentes

### AC5 — Isolamento por usuário

- `user_id` como FK único — relation 1:1 com `users`
- Server Action verifica `session.user.id`
- Tentativa de editar perfil sem autenticação retorna erro 401
