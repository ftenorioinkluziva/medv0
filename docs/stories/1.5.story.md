## Story

**Como** novo paciente,
**Quero** ser guiado por um onboarding simples ao criar minha conta,
**Para** que entenda o que o app faz e forneça as informações mínimas para a primeira análise.

---

## Acceptance Criteria

1. Step 1: Tela de boas-vindas com disclaimer explícito — aceite obrigatório para continuar
2. Step 2: Formulário com campos obrigatórios do Medical Profile básico (Story 1.3)
3. Step 3: Convite para primeiro upload de exame — pode ser pulado
4. Onboarding exibido apenas uma vez (`onboardingCompleted = true` após conclusão)
5. Após onboarding, usuário redirecionado para o dashboard
6. Usuário que já completou onboarding é redirecionado direto para o dashboard ao fazer login
