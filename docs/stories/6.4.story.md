# Story 6.4 — Gestão de Usuários (Admin)

**Status:** Ready for Review
**Linear:** BLA-55
**Branch:** —
**Cycle:** 1 (30/03–06/04/2026)

---

## Story

**Como** admin,
**Quero** visualizar usuários cadastrados, suas atividades e desativar contas quando necessário,
**Para** gerenciar o acesso à plataforma e monitorar a utilização por usuário.

---

## Acceptance Criteria

### AC1 — Listagem de usuários

- Rota `/admin/users` exibe tabela com: email, role, total de exames, total de análises, status (ativo/inativo), data de criação
- Tabela ordenada por `createdAt` desc
- Contador "Total de usuários" no header da página

### AC2 — Filtro por role

- Select de filtro com opções: "Todos", "patient", "admin"
- Filtro aplicado client-side, sem re-fetch
- Filtro e busca combinados funcionam simultaneamente

### AC3 — Desativação de conta

- Switch (ativo/inativo) em cada linha abre `AlertDialog` de confirmação antes de desativar
- Ao confirmar desativação: atualiza `isActive = false` na tabela `users`
- Usuário desativado não consegue fazer login (middleware/auth já verifica `isActive`)
- Após toggle: revalidar `/admin/users`, exibir toast de sucesso
- Erro no toggle exibe toast de erro
- Reativação funciona da mesma forma (sem AlertDialog — apenas confirmação para desativar)

### AC4 — Navegação no admin nav

- Link "Usuários" no `AdminNav` (já existe: `href: '/admin/users'`) navega para a rota correta

### API / Server Actions

Não há endpoints REST novos — tudo via Server Actions (`'use server'`).

**Toggle action:**
```typescript
// src/app/admin/users/_actions/users.ts
'use server'
import { auth } from '@/lib/auth/config'

export async function toggleUserActiveAction(
  id: string,
  isActive: boolean,
): Promise<ActionResult> {
  const session = await auth()
  if (session?.user?.id === id) {
    return { error: 'Não é possível desativar a própria conta.' }
  }
  const user = await getUserById(id)
  if (!user) return { error: 'Usuário não encontrado' }

  try {
    await db.update(users).set({ isActive, updatedAt: new Date() }).where(eq(users.id, id))
  } catch {
    return { error: 'Erro ao atualizar usuário. Tente novamente.' }
  }

  revalidatePath('/admin/users')
  return { success: true }
}
```
