## Risk Coverage

| Risco | Cenários | Mitigação |
|-------|----------|-----------|
| MEDIUM: API keys ausentes | 9.1-UNIT-006, 9.1-UNIT-007 | Erro claro para Anthropic; fallback Google |
| LOW: diferenças entre providers | 9.1-INT-003 | Backward compat para Google; outros via mock |

## Recommended Execution Order

1. P0 Unit (9.1-UNIT-001..006) — fail fast
2. P0 Integration (9.1-INT-001, 002) — wiring
3. P1 Integration (9.1-INT-003..005) — compat + orchestrators
4. P1 Unit (9.1-UNIT-005, 007) — edge cases
5. P1 E2E (fora do gate atual)
